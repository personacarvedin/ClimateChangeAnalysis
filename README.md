# ClimateChangeAnalysis

ClimateChangeAnalysis is an interactive notebook project focused on exploring and analyzing climate change data using PySpark. The project is designed for educational and analytical purposes, leveraging the scalability of PySpark for big data processing. The analysis can be run directly in Google Colab for convenience and accessibility.

## Features

- Data import, cleaning, and preprocessing with PySpark
- Exploratory Data Analysis (EDA) on climate change datasets
- Visualization of trends and patterns in climate data
- Fully runnable in Google Colab (no local setup required)
